# truffleJS
Like TruffleHog but for JavaScript

## Usage

```
% python3 main.py -h                                    
usage: main.py [-h] [-l LISTFILE] [-u URL] [-d DISABLE_ENTROPY]

optional arguments:
  -h, --help            show this help message and exit
  -l LISTFILE, --listfile LISTFILE
                        List file which contain list of URLs to be scanned for secrets.
  -u URL, --url URL     Enter the URL in which you want to find secrets.
  -d DISABLE_ENTROPY, --disable_entropy DISABLE_ENTROPY
                        Disable high entropy checks.

```
